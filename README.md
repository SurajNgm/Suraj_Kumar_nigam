# TANDEMLOOP Screening Test – Suraj Kumar Nigam
**Language Used:** Java  
**IDE:** VS Code 

---
### Problem-1
A calculator program using class and methods for Addition, Subtraction, Multiplication, and Division.

### Problem-2
Generates a series of odd numbers up to `x`.

### Problem-3
Generates a modified odd series depending on whether the input is even or odd.

### Problem-4
Counts how many numbers in a given list are multiples of 1–9.

---

### How to Run

Open the terminal in the project folder and run:

```bash
javac Problem-1.java
java Calculator

javac Problem-2.java
java Problem2

javac Problem-3.java
java Problem3

javac Problem-4.java
java Problem4
